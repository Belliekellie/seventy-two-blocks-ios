# 72 Blocks - Native iOS/Mac/iPad App

Native SwiftUI app for the 72 Blocks time management system.

## Overview

This is the native Apple platform version of 72 Blocks, built with SwiftUI for seamless experience across iPhone, iPad, and Mac.

## Backend

Uses the same Supabase backend as the web app:
- Authentication
- Block data storage
- Real-time sync

## Requirements

- Xcode 15+
- iOS 17+ / macOS 14+
- Swift 5.9+
